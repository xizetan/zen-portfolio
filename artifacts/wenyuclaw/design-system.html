Logo 系统

LOGO SYSTEM · ASSETS · DUAL VERSION · DECISION

文语标识由"开合双引号"构成——左上一对蓝色引号（Primary Blue，#1E5FBF）代表「人」的提问，右下一对 AI 绿引号（#00C9A7）代表「AI」的回响。本章节分三部分：01.1 标识资产与结构 · 01.2 双版本尺寸体系 · 01.3 使用决策。

暗色模式

DARK MODE · 重绘而非反转

暗色模式不是简单反转——简单反转会刺眼、饱和度过高、层级丢失。规则：品牌色降饱和（500→300）、背景明暗颠倒、卡片比页面亮一级（不用纯黑）、文字保持 AA 对比、边框稍亮。

03.6

六大常见错误

PITFALLS
✗ 01主色当信息色用
品牌被信息海淹没。蓝色提示请用 info 色系。
✗ 02辅色泛滥
稀释"AI 专属"认知。青绿严格只给 AI 元素。
✗ 03跨色系取色
红底配蓝字，破坏色彩协调。
✗ 04用纯黑做文字
#000 + 白底刺眼。用墨 700 #0A2540 替代。
✗ 05暗色简单反转
主色直接搬深底，饱和度过高刺眼。
✗ 06自创色阶
"我觉得这个蓝好看"。新色必须从现有色阶取。

RULES · 黄金法则

三条不可破的规则

辅助图形是背景，不是主角。触犯任何一条，都意味着整套系统失效。
01
STATIC VS DYNAMIC

静态选 A · 动态选 C

印刷 / PPT / 海报 → A 系统；产品 / 交互 / 数据 → C 系统。不要混用。
02
ONE PER SCREEN

同屏仅一种 · 避免混用

一个版面只用 A 或 C，不同时出现；防止视觉语言冲突。
03
NEVER DOMINANT

永远是配角

A 系统不透明度锚定 8–20%；C 系统动效时长锚定 1.2–2s。超出即喧宾夺主。
E · EXTENDED
